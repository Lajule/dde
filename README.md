# dde

A Dwight D. Eisenhower matrix application written using [webview][1].

![dde](https://github.com/Lajule/dde/blob/master/dde.gif)

[1]: https://github.com/webview/webview

## Usage

```sh
./dde -h
Usage of ./dde:
  -d	Debug mode
  -f string
    	Tasks filename (default ".dde.json")
```

## Build

```sh
go generate
go build
```
